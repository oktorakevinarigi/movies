# Introduction

Sehatq web monorepo design using turborepo features. There ares some packages and apps:

- `movie`: Movie web application powered by [Next.js](https://nextjs.org)
- `@/components`: docs, user interfaces and domain-spesific features powered by [Storybook](https://storybook.js.org/)
- `@/constants`: shared Constants
- `@/utils`: shared React, generic type and function utilities

The packages and apps source code in project strcuture seperti

```
  root
  ├── src
  |   ├── app                             # Create pages
  |   |   ├── example
  |   |   |   └── page.tsx                # Initial name page with route /example
  |   |   |
  |   |   ├── globals.css                 # Global Styles
  |   |   ├── layout.tsx                  # Initial Layout
  |   |   └── page.tsx                    # Page root 
  |   |
  |   ├── components           
  |   |   ├── features                        # Domain-specific features
  |   |   |   ├── feature-name
  |   |   |   |   ├── stories                 # Docs-specific react components
  |   |   |   |   |   └── c-n.stories.tsx     # Docs-specific react components
  |   |   |   |   |
  |   |   |   |   ├── component-name.tsx      # Docs-specific react components
  |   |   |   |   |
  |   |   |   |   ├── f-n-atoms.tsx           # Docs-specific react components
  |   |   |   |   |
  |   |   |   |   ├── f-n-constants.tsx       # Docs-specific react components
  |   |   |   |   |
  |   |   |   |   ├── f-n-model.tsx           # Docs-specific react components
  |   |   |   |   |
  |   |   |   |   ├── f-n-queries.tsx         # Docs-specific react components
  |   |   |   |   |
  |   |   |   |   └── index.tsx               # Reexport React component from file-name.tsx
  |   |   |   |
  |   |   |   └── index.tsx                   # Reexport React component from file-name.tsx
  |   |   |
  |   |   └── user-interfaces                 # Specific React, generic type and function utilities
  |   |
  |   ├── docs                                # Pure documentation pages
  |   |   ├── component
  |   |   |   ├── component-name.tsx          # Docs-specific react components
  |   |   |   |
  |   |   |   └── index.tsx                   # Reexport React component from file-name.tsx

```
