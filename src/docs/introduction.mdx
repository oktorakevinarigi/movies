# Introduction

Sehatq web monorepo design using turborepo features. There ares some packages and apps:

- `movie`: Movie web application powered by [Next.js](https://nextjs.org)
- `@/components`: docs, user interfaces and domain-spesific features powered by [Storybook](https://storybook.js.org/)
- `@/constants`: shared Constants
- `@/utils`: shared React, generic type and function utilities

The packages and apps source code in project strcuture seperti

